# 抖音自动加关注 Chrome 扩展

## 功能特点

- ✅ 根据关注名单自动关注抖音用户
- ✅ 可设置关注间隔时间（秒）
- ✅ 自动标记已关注用户，避免重复关注
- ✅ 中断后重新执行自动跳过已关注用户
- ✅ 密钥过期时间检查（通过网络API获取时间）
- ✅ 双时间API备用机制

## 安装方法

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启"开发者模式"（右上角）
3. 点击"加载已解压的扩展程序"
4. 选择本扩展的文件夹

## 使用方法

1. 登录抖音账号（在浏览器中）
2. 点击扩展图标打开弹窗
3. 设置密钥过期时间（Unix时间戳）
4. 设置关注间隔（秒）
5. 输入关注名单（每行一个用户URL）
6. 点击"开始关注"按钮
7. 在抖音页面等待自动执行

## 注意事项

- 需要在抖音页面（https://www.douyin.com/*）使用
- 确保已登录抖音账号
- 密钥过期后将无法继续使用
- 已关注的用户会自动跳过

## 文件说明

- `manifest.json` - 扩展配置文件
- `popup.html/css/js` - 扩展弹窗界面
- `content.js` - 内容脚本（在抖音页面执行）
- `background.js` - 后台服务脚本
- `icon.png` - 扩展图标（需要自行添加）
